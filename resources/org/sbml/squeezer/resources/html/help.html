The program SBMLsqueezer generates kinetic rate equations for models in SBML format. SBMLsqueezer is applied to the model currently loaded into the program, or, if used in plug-in mode, into the program CellDesigner.

Standard kinetics

If the standard kinetics should be applied, just click the apply button. This generate the respective kinetics the current SBML model.

Options

To show the options of the SBMLsqueezer just click on “show options”. The dialog boxs allows the specification of different kinetic settings for the current SBML model.
