Installation Instructions

For detailed explanations how to install, maintain and use SBMLsqueezer, the Tutorial and the Kinetic Law Supplement are very helpful as these documents provide a comprehensive introduction and description of the program. Section Quick Tour of this web page provides a quick introduction into the usage of SBMLsqueezer. To integrate SBMLsqueezer as an equation generating core into your application, the API documentation gives the necessary information.

  • Requirements:
    • Minimal computer: CPU 800 MHz, 128 MB RAM, 4 MB free disk space.
    • Java VM 1.5 or later (available from Sun),
    • For the plug-in mode: CellDesigner 4.0α or higher (available from CellDesigner)
  • Download SBMLsqueezer from the download section of this web page
  • If you want to use SBMLsqueezer as a plug-in for CellDesigner, please procede as follows:
    • Install CellDesigner (see the CellDesigner web site for further instructions)
    • Copy the SBMLsqueezer jar-file into the plugin directory of CellDesigner. For instance
      /opt/CellDesigner4.1beta/plugin/under Linux,
      /Applications/CellDesigner4.1beta/plugin/under Mac OS X, or
      C:\Programs\CellDesigner4.1beta\plugin\under Windows,
    Please not that this works for all SBMLsqueezer versions.
  • For the stand-alone mode (requires at least SBMLsqueezer version 1.3), you can copy the SBMLsqueezer JAR file to any directory in your system. You can start the application by double clicking on the JAR file. It might be necessary to install libSBML version 4.0.0 before using SBMLsqueezer in stand-alone mode. If it does not run after a double click, please set the path for libSBML for your respective system. To this end, please see the instructions on the libSBML homepage. It might be necessary to start SBMLsqueezer from the console, e.g., by typing:
    java -cp SBMLsqueezer.1.3.jar:<path to libSBML> org.sbml.squeezer.SBMLsqueezer
    A list of all currently available command-line options can be found in this overview. The example start script run.sh may demonstrate how to configure your system appropriately.

Example start script for SBMLsqueezer 1.3

Here you can download the example start script run.sh, which demonstrates how to set up the necessary environment variables for a Linux operating system. For your particular platform it may be necessary to modify this script, but it may serve as a good start point and ease invoking SBMLsqueezer 1.3 in its stand-alone mode.
