JSBML for CellDesigner 4.3

In order to use SBMLsqueezer 2.0 as a plug-in for CellDesigner 4.3, you will also need to download a special JSBML package here: JSBML 1.0β1 with dependencies.

Follow the installation instructions in the Users' Guide (see below). Please note, that this file's name has been choosen to simplify the installation, the file name does not match the version of JSBML, but this name is required for CellDesigner to run.
