Generating kinetic equations for biochemical networks


Quick Tour

Adjusting the Preferences

The options in this menu are explained in detail in the Tutorial. This page gives only a short summary of the most important options. By clicking “Generate” all settings are applied and the kinetic equations will be generated for the whole system and presented in a table. Afterwards, you can go back and find your settings again.

The configuration of SBMLsqueezer is available in the CellDesigner plug-in menu, in the Edit menu of the SBMLsqueezer stand-alone version, and also when creating rate equations in one go.

Kinetics settings

This dialog provides the basic configuration for the rate law generation.
  • General Options: Here you can specify if
    • SBMLsqueezer should automatically set the boundary condition flag for species that represent gene coding elements. This can be necessary because often these species are neither consumed nor produced during a transcription/translation reaction.
    • SBMLsqueezer should remove all parameters and unit objects that are not referenced by any equation or other SBML construct anymore. It may happen that after creating new rate laws for the whole network praameters that where defined previously are no longer referenced. However, you may want to keep these values.
    • catalysts must be added to each reaction explicitly or if for sake of simplicity all reactions are considered to be enzyme-catalyzed.
    • warnings should occur if one reaction has an unrealistic large number of participating reactants. By default at least three reactants is considered unrealistic.
    • The last three options allow you to specify the default value for compartments, species, and parameters. This is important for cases, in which the size of certain compartments or the initial amount or concentration of a species has not been set right now. All newly generated parameters are also set to the default value. Not that in this step SBMLsqueezer does not change the units or spatial dimentions of the element.
  • Generate New Kinetics: By default SBMLsqueezer only creates new kinetic equations for reactions, which currently don't have such a law. You can also force to overwrite all existing kinetics by selecting “For all reactions”.
  • Reversibility: If you want to set all reactions to reversible and create the kinetic laws accordingly, select the default “Model all reactions in a reversible manner”. Otherwise SBMLsqueezer can also keep the reaction irreversible or reversible, respectively, as given in the model.
  • Version of modular rate laws: Here you can specify if the cat, hal, or weg version of the modular rate laws should be used. The cat version is the simplest one with the lowest number of parameters but may not lead to thermodynamically correct systems. The number of parameters and the thermodynamic correctness increases in the hal version but only the weg version ensures fully correct parameter dependencies.
  • Species to Be Treated as Enzymes: Since it is not always clear which catalyzer can act as an enzyme, this section allows you to select those species, that may be considered as enzymes when acting as a catalyzer. By default these are “Generic protein”, “RNA”, “Complex” and “Truncated protein”. You may also select “Receptor”, “asRNA”, “Unknown” or “Simple molecule” or deselect any one of the preselected types. A reaction catalyzed by a species, which is no enzyme, will be modeled using the generalized mass-action rate law.

Reaction Mechanisms

This section allows you to specify the type of kinetic equation to be applied for each specific type of reaction. Currently SBMLsqueezer distinguishes the following basic types of reactions: Non-Enzyme Reactions, i.e., reactions that are not catalyzed by an enzyme, Uni-Uni type, i.e., enzyme reactions with one reactant and one product, Bi-Uni type, i.e., enzyme reactions with two reactants and two products. Here the same species may occur with the stoichiometry coefficient two. Furthermore, Bi-Bi type (like Bi-Uni but with two products), and other enzyme reactions. The last case covers all those enzyme-catalyzed reactions that do not fit into any other case. Finally, there are several rate equations available for gene-regulatory reactions, i.e., translation and transcription. Please not that the selection of applicable default rate laws changes if in the dialog &ldqou;Kinetics settings&rdqou; the setting is changed that decides whether or not to model all reactions reversibly. Note that you are able to change the generated equations subsequently by clicking on the name of the reaction in the result table.

Program settings

Here you can specify the default paths for opening and saving files.

LaTeX output settings

This menu allows you to specify the properties of the LaTeX export, similar to the configuration of SBML2LaTeX. This function may be useful to gain a structured overview about all rate laws in the model. However, for a fully featured model report, we strongly recomment using SBML2LaTeX.
