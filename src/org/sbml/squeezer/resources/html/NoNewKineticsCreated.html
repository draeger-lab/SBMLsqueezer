No rate equations were created because the model already contains a rate law for each reaction. By default, SBMLsqueezer does not overwrite already existing rate laws. If you feel unsure of what to do, please click on the “Help” button below or visit SBMLsqueezer's homepage at http://www.ra.cs.uni-tuebingen.de/software/SBMLsqueezer.
