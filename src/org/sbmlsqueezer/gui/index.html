SBMLsqueezer

The SBMLsqueezer generates kinetic rate laws for SBML models. The SBMLsqueezer is applied to the model currently loaded in the CellDesigner.

Standard kinetics

If the standard kinetics should be applied, just click the apply button. This generate the respective kinetics the current SBML model and store the default settings file with the ".settings" extension. The SBML file without kinetics will be stored with the extension ".old.sbml".

Options

To show the options of the SBMLsqueezer just click on "show options". The dialog boxs allows the specification of different kinetic settings for the current SBML model.

Homepage

You can find further information at the SBMLsqueezer homepage.
